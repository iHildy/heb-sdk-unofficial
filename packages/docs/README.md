# HEB SDK (Unofficial) Docs

Template docs site cloned from the original design, with placeholder content and examples.

## Commands

All commands run from this folder:

| Command | Action |
| --- | --- |
| `pnpm install` | Install dependencies |
| `pnpm dev` | Run the local dev server |
| `pnpm build` | Build for production |
| `pnpm preview` | Preview the build |

## Structure

- `src/content/docs/` Markdown/MDX content
- `src/components/` Site components
- `src/assets/` Images and logos
