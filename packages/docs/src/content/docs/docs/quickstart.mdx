Start here to run your first SDK call.
Pick a session type before you code.

## Install

```bash
npm install heb-sdk
# or
pnpm add heb-sdk
```

---

## Create a session

Cookie sessions use logged-in browser cookies.
Bearer sessions use OAuth tokens and unlock mobile-only features.

**Cookie session**

```ts
import { createSessionFromCookies, HEBClient } from "heb-sdk"

const session = createSessionFromCookies(process.env.HEB_COOKIES ?? "")
const heb = new HEBClient(session)
```

**Bearer session**

```ts
import { createTokenSession, HEBClient } from "heb-sdk"

const session = createTokenSession({
  accessToken: process.env.HEB_ACCESS_TOKEN ?? "",
  refreshToken: process.env.HEB_REFRESH_TOKEN,
  idToken: process.env.HEB_ID_TOKEN,
  expiresIn: 1800
})

const heb = new HEBClient(session)
```

Cookie input can be a Cookie header string or DevTools JSON.
Use a bearer session for search, product details, homepage, orders, account, and weekly ad.

---

## Set a store

Store context drives pricing and availability.
Set it before search, product details, homepage, and weekly ad calls.

```ts
await heb.setStore("790")
```

---

## Run a call

Search uses the mobile API and needs bearer tokens.
Cart reads work with either auth mode.

```ts
const results = await heb.search("coffee", { limit: 10 })
const names = results.products.map((product) => product.name)
console.log(names)
```

```ts
const cart = await heb.getCart()
console.log(cart.itemCount, cart.total.formatted)
```
